# Project Alpha Launcher

A simple desktop launcher to play custom games with friends.


```bash
git clone https://github.com/gianmarcomichelini/project_alpha_launcher.git
cd project_alpha_launcher
npm install
```

## Dev
```bash
npm start
```


## Build

```bash
# Windows
npm run build:win
```

```bash
# macOS
npm run build:mac
```

Output will be in the `dist/` folder.
